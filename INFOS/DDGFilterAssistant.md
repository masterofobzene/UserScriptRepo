### USE CASE:
--------------
It adds "negative words" to your search box automatically when you click on the desired options, helping you 
filter the DDG search results.

### USAGE:
---------------
Click on the desired filter, a description of what they do follows:

```
NO BEST LISTS: Avoids "lists trash sites" e.g. "10 BEST IMAGE EDITORS".
NO AI: Removes AI based services e.g "our AI models remove noise...".
NO ONLINE APPS: Avoids online services e.g. "online image cropper".
ONLY PDF: Useful for searching for manuals or books.
GITHUB ONLY: Searches only in github.com
REDDIT ONLY: Searches only in reddit.com
NO LINUX: Avoids results showing support pages with linux discussions.
NO APPLE: Same than above, it extends the filter to IOS devices too.
NO ANDROID: Filters android results e.g "download {something} APK for free".
```
If you want to clear all checkboxes, use the broomstick icon 🧹

### WHY?:
---------------
Search engines today NEED negative words to filter results, as they get more and more bloated with ads and 
fillers and trash results. DDG is terrible as it uses Bing and Bing is the worst search engine regarding relevance in results. 
This extension just makes life easier to the users so they don't have to use the special syntax to 
search or to the ones like me that need to use these filters recurrely.
