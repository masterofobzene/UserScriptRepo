### USE CASE:
--------------
Lazy game discovery. This userscript will add 3 new buttons to Gog.com; one to search for non(voice)commented gameplay videos
on youtube to roughly judge gameplay, one for searching that game on torrminatorr and another for searching on cs.rin ru forum 
for download. 
ATTENTION: IT IS NOW REQUIRED TO HAVE A USER ON TORRMINATORR AND/OR CS.RIN.RU TO WORK WITH THIS EXTENSION.

### USAGE:
---------------
Just click the buttons.

2.1: Added a "search on cs.rin.ru" button to also search for that game on that forum for the steam version, since it seems torrminatorr has a lot of dead links.

### WHY?:
---------------
It is useful to use gog.com as a catalog; it makes all the process of finding games easier.
