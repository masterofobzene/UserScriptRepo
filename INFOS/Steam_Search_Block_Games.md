### USE CASE:
--------------
Block tag-abused games from appearing.

### USAGE:
---------------
On each game card a red "block" button appears so you can hide it if you want effectively taking care of 
shitty games that use tags you like or need to keep in the search.

### WHY?:
---------------
Tag abuse is a widespread normal bad practice nowadays. If you think, you will realize 70% of the games have for example
"simulation" tag, one would think these are aircraft sims, or racing sims, or military sims, but no, you can get
"Walking simulator" or "Grass Simulator" or "Goat Simulator". It is also present on "normal" games like "Colony builders"
too, making this a complete mess. Everything is considered a simulation nowadays but this is not aimed at that tag only,
it helps with all the content available. Tag abuse made the search system useless basically. Removing Simulator tag will 
remove 70% of the store. This is where this script comes to aid the abused user, letting him/her remove trash surgically
without removing games they may want to see.
